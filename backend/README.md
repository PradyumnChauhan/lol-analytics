# Backend Server

Express.js backend server for LoL Analytics.

## Setup

1. Install dependencies:
```bash
npm install
```

2. Copy `.env.example` to `.env`:
```bash
cp .env.example .env
```

3. Configure environment variables in `.env`

4. Start the server:
```bash
npm start
# or for development with auto-reload:
npm run dev
```

## Environment Variables

See `.env.example` for all available environment variables.

## Configuration

- Server configuration: `.env` file
- Riot API credentials: `config/riot-config.json` (auto-created)

## API Endpoints

See the main README or server logs for available endpoints.
